# Canvas Assignment Manager
This is a CLI tool that allows you to view and submit assignments to Canvas from the terminal. It is made with GO.

# Prerequistites
Git and GO programming languages must be installed before use, use these provided links to understand how to get these for your OS.

For Git:
https://git-scm.com/install/

For GO:
https://go.dev/dl/

# Setup Instructions
Clone this repository in a desired location with the following command:
